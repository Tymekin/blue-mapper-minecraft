# Area Player Scanning
A Figura + BlueMap script that informs you whenever a player enters or leaves designated rectangle area in the Overworld, set using X and Z coordinates of two corners.

Requires the Auria HTTP Library available at https://github.com/lua-gods/figura-libraries/blob/main/auria%20http%20library/http.lua (only http.lua file)

Configure by tweaking the variables inside, might need to be modified in case of your server using non-standard Blue Map file directories. Make sure to have Networking enabled and Whitelist disabled in your Figura settings.
